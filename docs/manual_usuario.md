# 📘 Manual de Usuario – THE FORUM

## 1. Introducción
THE FORUM es una plataforma web de foros orientada inicialmente a videojuegos, donde los usuarios pueden crear, participar y gestionar discusiones sobre distintos temas.

Este manual está dirigido a usuarios que no conocen la aplicación y explica cómo utilizar sus funcionalidades principales.

---

## 2. Acceso a la aplicación

Para acceder a THE FORUM, el usuario debe abrir la aplicación desde su navegador web.

No es necesario estar registrado para navegar por los contenidos públicos, pero sí para interactuar.

---

## 3. Página principal

Al entrar en la aplicación, el usuario encontrará:

- Lista de videojuegos disponibles
- Foros destacados
- Barra de navegación superior

### Barra de navegación:
- Inicio
- Login / Registro
- Perfil (si el usuario está autenticado)
- Crear foro

---

## 4. Registro de usuario

Para registrarse:

1. Pulsar en el botón **"Registro"**
2. Rellenar el formulario con:
   - Nombre de usuario
   - Email
   - Contraseña
3. Confirmar el registro

Una vez registrado, el usuario podrá iniciar sesión.

---

## 5. Inicio de sesión

Para iniciar sesión:

1. Pulsar en **"Login"**
2. Introducir:
   - Email
   - Contraseña
3. Acceder a la cuenta

El sistema autenticará al usuario y le dará acceso a funcionalidades completas.

---

## 6. Perfil de usuario

Desde el perfil, el usuario puede:

- Ver su información personal
- Editar su perfil
- Consultar sus publicaciones
- Cerrar sesión

---

## 7. Navegación por videojuegos y foros

La estructura de la aplicación es:

Videojuego → Foro → Publicaciones

### Pasos:
1. Seleccionar un videojuego
2. Acceder a un foro
3. Visualizar las publicaciones existentes

---

## 8. Crear un foro

Los usuarios registrados pueden crear foros.

### Pasos:
1. Pulsar en **"Crear foro"**
2. Introducir:
   - Título
   - Descripción
   - Videojuego asociado
3. Confirmar creación

---

## 9. Publicar mensajes

Dentro de un foro, los usuarios pueden:

- Crear publicaciones
- Responder a otras publicaciones (hilos)

### Crear publicación:
1. Pulsar en **"Nuevo mensaje"**
2. Escribir el contenido
3. Publicar

### Responder a una publicación:
1. Seleccionar un mensaje
2. Pulsar en **"Responder"**
3. Escribir respuesta

---

## 10. Gestión de publicaciones

Los usuarios pueden:

- Editar sus propios mensajes
- Eliminar sus publicaciones

Los administradores y moderadores tienen permisos adicionales.

---

## 11. Roles de usuario

Existen distintos tipos de usuario:

- Invitado: solo puede visualizar contenido
- Usuario registrado: puede participar
- Moderador: puede gestionar contenido
- Administrador: control total del sistema

---

## 12. Contacto y soporte

La aplicación dispone de un formulario de contacto donde el usuario puede:

- Enviar sugerencias
- Reportar errores
- Notificar contenido inapropiado

---

## 13. Recomendaciones de uso

- Utilizar contraseñas seguras
- Respetar las normas de la comunidad
- No publicar contenido inapropiado

---

## 14. Compatibilidad

THE FORUM es compatible con:

- Google Chrome
- Mozilla Firefox
- Microsoft Edge

También dispone de diseño adaptado para dispositivos móviles y tablets.

---

## 15. Cierre de sesión

Para cerrar sesión:

1. Acceder al perfil
2. Pulsar en **"Cerrar sesión"**

---
